# Haven 🏠

A faith-based small group PWA. Clean, warm, mobile-first. Built to be installed on iPhone.

## Features

- **Group Chat** — Text, prayer requests (🙏), and praise reports (✨)
- **Prayer Wall** — Share requests, pray for others, mark as answered
- **Weekly Check-ins** — Group accountability around a weekly focus question
- **PWA** — Installable on iOS/Android via "Add to Home Screen"
- **No backend** — All data stored in localStorage (great for demos)

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v4 (`@tailwindcss/vite`)
- framer-motion
- lucide-react
- react-router-dom

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Design

- Background: `#FAFAF5` (warm cream)
- Navy: `#1E3A5F`
- Gold: `#C9A853`
- Mobile-first, bottom tab navigation
- Clean and minimal — not religious kitsch
